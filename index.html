Module 1: What is Misinformation and Disinformation?

Misinformation refers to false or inaccurate information shared without intent to deceive.

Disinformation refers to false information shared deliberately to mislead others.

Module 3: Using Tools to Verify Content

Use Google Reverse Image Search to track image sources.

Visit fact-checking websites like Snopes, PolitiFact, or FactCheck.org.

Use TinEye to trace image edits and versions.

Cross-verify content across multiple reputable sources.

Module 4: Evaluating Post Credibility

Use this checklist to assess credibility:

  • Who: Is the author credible?
  • What: Is the source reliable?
  • When: Is the info current?
  • Where: Are sources cited?
  • Why: Is the intent to inform or mislead?
